Campaigns report
Fetching stats for advertiser campaigns.

curl "https://api.trafficstars.com/v1.1/advertiser/custom/report/by-day" \
  -H "Authorization: bearer your_access_token" \
  --get \
  -d "campaign_id=1" \
  -d "date_from=2015-01-01" \
  -d "date_to=2015-01-01"

HTTP Request
> GET /v1.1/advertiser/custom/report/by-{group_type}

Advertiser must replace {group_type} with one of the available group types

Query parameters
Parameter	Data type	Options	Description
date_from	string	required	date from. appropriate formats: YYYY-MM-DD or YYYY-MM-DDTHH:00:00Z (except stats by subid)
date_to	string	required	date to. appropriate formats: YYYY-MM-DD or YYYY-MM-DDTHH:00:00Z (except stats by subid)
app_id	[]number		expected sites
exclude_app_id	[]number		excluded sites
banner_id	[]number		expected banners
browser_id	[]number		expected browsers
campaign_id	[]number		expected campaigns
carrier_id	[]number		expected carriers
country_code	[]string		expected countries in 2-symbols format ISO 3166-1
device_id	[]number		expected devices
format_id	[]number		expected formats
language_code	[]string		expected languages in 2-symbols format ISO 639-1
os_id	[]number		expected operating systems
rtb_client_id	[]number		expected RTB clients
spot_id	[]number		expected spots
exclude_spot_id	[]number		excluded spots
url_id	[]number		expected resources
category_id	[]number		expected categories
network_traffic_type	number		get proxy/non-proxy traffic
offset	number		set offset of response items (use whit limit)
limit	number		limit number of response items
order_by	string	for subid ONLY	order by field name (example: "revenue" or "-revenue")
